# Placement-Eligibility-Streamlit-Application
Placement Eligibility Streamlit Application
## 📌 Project Overview
This project is designed to evaluate the placement readiness of students based on their academic performance, programming skills, soft skills, and project experience.  
The system uses data analysis, SQL database storage, and a Streamlit web interface to display results interactively.

---

## 🏫 Domain
**Data Science**

---

## ❓ Problem Statement
Training and placement departments face difficulties in manually tracking and shortlisting students based on multiple performance parameters.  
A data-driven system is required to automate eligibility assessment and improve decision-making.

---

## 💼 Business Use Cases
- Identify students who are ready for placement recruitment
- Track performance improvements over time
- Provide insights to training teams for planning activities
- Help students understand their strengths and weaknesses

---

## 📊 Dataset Description
| Column | Description |
|--------|-------------|
| Name | Student Name |
| Course Batch | Department / Batch |
| Programming Score | Score out of 100 |
| Soft Skills Score | Average communication & teamwork rating |
| Placement Status | Ready / Not Ready / Placed |

Dataset used: *Self-created dataset for analysis and system testing*
